# Проект: Елена Боброва — персональный веб-проект

## Что это за проект

Персональный веб-проект Елены Бобровой, созданный в процессе обучения веб-разработке.
Проект включает цифровую визитку и лендинг для привлечения партнёров в MLM-бизнес.
Все страницы написаны на чистом HTML и CSS, без фреймворков — это первые практические результаты.

---

## Для кого

- **Потенциальные партнёры** — люди, которые хотят дополнительный доход без стартовых вложений
- **Личный бренд** — визитка для знакомства и быстрого выхода на связь

## Стиль дизайна

Проект использует два стиля — по одному на каждую страницу:

### Визитка — светлый, воздушный
- Белая карточка с мягкими тенями
- Лавандово-голубая цветовая палитра
- Анимированные ambient-пятна на фоне
- Плавные переходы и эффект shimmer на имени

### Лендинг — тёмный, технологичный (glassmorphism)
- Тёмный фон `#06040f` с сеткой и частицами
- Стеклянные карточки с backdrop-filter blur
- Градиенты фиолетовый → синий (`#8b5cf6` → `#3b82f6`)
- Анимированная CTA-кнопка с пульсацией и shine-эффектом
